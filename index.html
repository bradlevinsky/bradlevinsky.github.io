Welcome to my Web Space

This is my portfolio that showcases the progress I have made during my INFO16206 Scripting and Web Languages class.

  • To explain a bit about this page, it contains a fixed header at the top of the screen. Within the header is my name and letting you know this is my web space. On the left, there is an icon that drops down a navigation menu that will allow you to navigate my web space. The menu will eventually include a scrollbar for easier navigation. Right now, there are two options: the About page and the Projects page. On the top right, I have an image of the Sheridan logo that is mapped, and when clicked, it will bring you to the Sheridan website.

  • The new items in my Projects Page include the PDF viewer, which uses an <iframe>, and a username/password form using <form>. The usable username and password are in a comment along with my initial assignment submission message.

  • In the About Page, I have added a gallery in the middle to list different movies. The sidebar contains hyperlinks that jump to the relevant section with the movie details. Clicking on the images opens them in a new tab, without navigating away from the current screen.

  • New styling features include using multiple classes, z-index, overflow-y, and other elements not explicitly covered in the lessons. I’ve added comments in the styling file and kept similar styling elements grouped together for easier readability.
